# 🎯 Nested Div Target Design (HTML + CSS)

This project demonstrates how to create **multiple nested circles (div inside div)** using pure HTML and CSS.

It is a simple target-style design created using:
- HTML5
- CSS3
- Flexbox
- Border Radius
- Hover Effects
- CSS Animations (optional)

---

## 🚀 Preview

Target style design made using nested div structure.

---

## 🏗️ Project Structure
outer
└── inner
└── both


- `outer` → Main circle  
- `inner` → Second circle  
- `both` → Center circle  

---

## 💡 Concepts Used

✅ Nested Div (Div inside Div)  
✅ Flexbox Centering  
✅ Border-radius: 50%  
✅ CSS Hover Effects  
✅ CSS @keyframes Animation
